HTML (HYPERTEXT MARKUP LANGUAGE)

HTML (Hypertext Markup Language) is the standard markup language used to create and design web pages. It defines the structure and layout of a web document by using a variety of tags and attributes. HTML is one of the core technologies of the World Wide Web, alongside CSS and JavaScript.

JavaScript

JavaScript JavaScript is a dynamic, lightweight programming language primarily used for creating interactive and dynamic content on web pages. It is one of the core technologies of the web, alongside HTML and CSS. JavaScript enables developers to implement features such as real-time updates, animations, form validations, and multimedia control, making websites more engaging and functional.

CSS

CSS CSS is acronym of Cascading Style Sheets. It helps to define the presentation of HTML elements as a separate file known as CSS file having .css extension.

Tim Berners- Lee

Tim Berners- Lee Tim Berners-Lee invented the World Wide Web in 1989 while working at CERN, the European Particle Physics Laboratory. His invention aimed to facilitate automated information-sharing among scientists globally. He also developed the first web client and server, and his specifications for URIs, HTTP, and HTML were foundational to the web's growth.
